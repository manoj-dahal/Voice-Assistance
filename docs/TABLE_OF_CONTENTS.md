## Documentation status rules

- **Available** — implemented and verifiable.
- **Adapter-ready** — boundary exists; connector is not installed.
- **Planned** — practical roadmap work.
- **Research** — requires substantial validation or specialized integration.
- **Speculative** — concept only.
- **Simulation** — preview without external execution.
